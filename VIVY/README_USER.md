## VIVY 功能说明书（桌宠版）

这份文档面向**使用者**与**轻度开发者**：介绍 VIVY 能做什么、怎么用、怎么配置，以及遇到问题时如何排查。

---

## 1. VIVY 是什么

VIVY 是一个运行在本机的桌面桌宠（PyQt6 + Flask + SQLite），提供：

- **日常聊天**：普通聊天/创作模式两种风格
- **图片理解（可选）**：聊天附带图片，让模型“看图”回复
- **记忆模块（可编辑）**：保存偏好与摘要，影响后续对话
- **沉浸写作**：大屏写作窗口 + 选区/全文的流式辅助（润色/续写/点评/加强）
- **语音**：语音输入与回复朗读（按配置）

大模型调用走 **OpenAI 兼容**的 `/chat/completions` 接口（默认配置为 DeepSeek，但也可换成其他兼容服务商）。

## 2. 快速开始

### 2.1 启动方式

你可能有两种启动方式：

- **开发方式运行**：直接运行 `desktop_pet.py`
- **发布版运行**：双击 `VIVY.exe`（如果你拿到的是打包版）

### 2.2 首次配置（必须）

在 `VIVY/.env` 中至少配置：

- `DEEPSEEK_API_KEY=你的Key`

你可以复制 `VIVY/env.example` 为 `.env` 作为起点。

## 4. 功能说明

### 4.1 普通聊天 / 创作模式

- **普通聊天（chat）**：更快、更短、更日常
- **创作模式（creative）**：更发散、更长、更适合灵感与写作（一般更慢）

你可以通过界面上的“形态”按钮切换。

### 4.2 今日灵感

点击“今日灵感”会返回结构化内容（示例）：

- 发现
- 联想
- 邀请问题

适合快速打开话题或获得写作素材。

### 4.3 脑洞短剧

点击“脑洞短剧”会让 VIVY 生成一段短小的即兴小剧场（3–8 行），偏轻松脑洞风格。

### 4.4 图片理解（可选，多模态）

你可以在发送消息时附带一张图片，让模型结合图片内容回复。

使用方法：

1. 点击 `🖼` 选择图片（出现 `🖼✓` 表示已选中）
2. 输入你的问题（例如“这张图里写了什么？”、“这张截图说明了什么问题？”）
3. 发送

重要说明：

- 图片理解需要**支持视觉的模型**。如果没配置或模型不支持，VIVY 会提示你切换/配置视觉模型。
- 图片请求通常比纯文本慢，必要时需要增加超时时间（见参数表）。

### 4.5 沉浸写作（大屏 + 流式辅助）

沉浸写作窗口用于长文编辑，并可对选区/全文进行流式辅助：

- `polish`：润色
- `continue`：续写
- `critique`：点评
- `improve`：加强
- `free`：自由发挥

### 4.6 文档创作辅助（读取文档）

右键菜单“读取文档（创作辅助）”可加载：

- `.txt` / `.md` / `.docx` / `.pdf`

并将内容作为创作参考，让模型在对话中更贴合上下文。

### 4.7 语音

常见交互：

- **语音输入**：点击 🎤 进行语音识别，识别到文本后会填入并自动发送
- **语音回放**：可开关回复朗读（按配置）

## 5. 配置参数表（.env）

下面列出常用环境变量（复制 `env.example` 生成 `.env` 后按需修改）。

### 5.1 大模型（OpenAI 兼容 /chat/completions）

| 变量名 | 作用 | 默认值/示例 |
|---|---|---|
| `DEEPSEEK_API_KEY` | API Key（必填） | `sk-...` |
| `DEEPSEEK_BASE_URL` | API Base URL | `https://api.deepseek.com` |
| `DEEPSEEK_MODEL` | 文本聊天模型 | `deepseek-chat` |
| `DEEPSEEK_VISION_MODEL` | **视觉模型**（看图用） | 例如 `gpt-4o-mini` / 服务商的视觉模型名 |

### 5.2 超时/性能

| 变量名 | 作用 | 默认值 |
|---|---|---|
| `VIVY_CHAT_TIMEOUT` | 普通聊天超时（秒） | 快速模式 25 / 非快速 60（由程序决定） |
| `VIVY_CREATIVE_TIMEOUT` | 创作模式超时（秒） | `75` |
| `VIVY_IMAGE_TIMEOUT` | **带图请求超时（秒）** | `90`（建议 90–180） |
| `VIVY_FAST_MODE` | 快速模式（更快更短） | `true` |

### 5.3 图片限制

| 变量名 | 作用 | 默认值 |
|---|---|---|
| `VIVY_IMAGE_MAX_BYTES` | 图片最大字节数 | `3000000` |

### 5.4 本地服务

| 变量名 | 作用 | 默认值 |
|---|---|---|
| `FLASK_PORT` | 本机 API 端口 | `5000` |

### 5.5 记忆/待机

| 变量名 | 作用 | 默认值 |
|---|---|---|
| `LLM_SUMMARY` | 是否启用 LLM 摘要写入记忆 | `true` |
| `VIVY_IDLE_TIMEOUT` | 空闲多少秒进入待机收起 | `30`（你的 `.env` 可能已改） |

## 6. FAQ（常见问题）

### 6.1 没有回复 / 一直“思考中”

优先检查：

- `.env` 是否存在、`DEEPSEEK_API_KEY` 是否正确
- 网络是否能访问 `DEEPSEEK_BASE_URL`
- 是否触发了代理/SSL 问题（见 6.4）

### 6.2 记忆模块点不了 / 文本框无法输入

可能原因：

- 被某些浮层特效挡住鼠标事件（新版本已修复：浮层不再拦截点击）
- 正在处理请求导致 UI 暂时禁用（busy 状态）

建议：

- 等待当前请求结束
- 重启程序后再试

### 6.3 图片识别超时

图片请求通常更慢，建议：

- 配置 `DEEPSEEK_VISION_MODEL`
- 把 `VIVY_IMAGE_TIMEOUT` 设置为 90–180 秒
- 降低图片大小（或截图裁剪）

### 6.4 代理/SSL 报错（ProxyError / SSLEOFError / unexpected eof）

常见原因是系统代理影响了网络请求（尤其是 Git/Pip/某些 HTTPS 请求）。

处理思路：

- 临时关闭系统代理后重试
- 或在对应工具/代码中明确禁用代理
- 确认代理软件对目标域名放行/直连

### 6.5 操作指南弹窗文字看不清

新版本已为操作指南弹窗添加深色主题样式；如果你仍遇到显示异常，请升级到最新版本或反馈截图。

## 7. 发布与打包（给维护者）

仓库提供脚本 `VIVY/build_release.ps1`：

- 会创建/复用 `.venv`
- 安装依赖与 PyInstaller
- 生成图标 `release/vivy.ico`
- 打包生成：
  - `dist/VIVY-release/`（文件夹）
  - `dist/VIVY-release-v<Version>.zip`（压缩包）

Inno Setup 安装器脚本参考：`VIVY/release/VIVY.iss`
